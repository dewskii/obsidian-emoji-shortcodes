# Emoji Shortcodes

Slack-like emoji support for ObsidianMD.

> [!NOTE]This is a fork of [obsidian-emoji-shortcodes](https://github.com/phibr0/obsidian-emoji-shortcodes) by phibr0.

## Features

- Type `:shortcode:` to insert emojis (e.g., `:joy:` → 😂)
- **Custom emojis** — upload your own images (PNG, GIF, JPEG, WebP)
- **Hover tooltips** — see the shortcode when hovering over any emoji
- Autocomplete suggester with emoji preview

## Custom Emojis

Add custom emojis via the ribbon button or command palette (`Add custom emoji`). Drag and drop an image or use the file picker, then give it a shortcode.

Custom emojis work just like built-in ones — type `:your_shortcode:` and select from the suggester. If you're itching for custom emoji's but don't know where to find them, [slackmojis](https://slackmojis.com/) or [emoji.gg](https://emoji.gg/) are good spots to find them.

## Settings

- **Immediate Emoji Replace** — Render emojis in Live Preview (on) or only in Reading mode (off)
- **Emoji Suggester** — Show autocomplete when typing `:`
- **History Priority** — Prioritize recently used emojis in suggester

## Installation

1. Go to **Settings → Community Plugins → Browse**
2. Search for the plugin name
3. Click **Install**, then **Enable**
